# OrderClick System

OrderClick - Point of Sale and Inventory Management System

A streamlined, web-based management solution designed for efficient sales processing, real-time inventory tracking, and order verification. OrderClick simplifies the checkout experience and provides administrative oversight for transaction records and product availability.

🎯 Features

Sales Processing – Intuitive checkout interface for handling customer orders.
Transaction Registry – Comprehensive logging of all sales with real-time status tracking.
Inventory Management – Real-time stock monitoring and product oversight.
Order Verification – Administrative workflow to verify, update, and manage the status of pending orders.
Advanced Filtering & Search – Search transactions by invoice or customer, and filter registry logs by specific date ranges.
Analytics Dashboard – Visual summary of average order sizes, total sales, and performance metrics.
Data Export – Generate and download formatted PDF reports of filtered sales data for accounting and auditing purposes.
Secure Authentication – Role-based access control (RBAC) to ensure system security and restricted admin access.

🛠️ Technology Stack

Frontend

React 18 - UI framework
TypeScript - Type safety
Vite - Build tool
Tailwind CSS - Styling
Axios - API requests
jsPDF & jspdf-autotable - Document generation

Backend

Node.js - Runtime
Express - Web framework
TypeScript - Type safety
PostgreSQL - Database
JWT - Authentication
CORS - Security configuration

🚀 Quick Start
1. Clone Repository
   
   git clone <your-repo-url>
   cd orderclick
   
2. Install Dependencies

# Install frontend dependencies
    npm install

# Install backend dependencies
    cd backend
    npm install
    cd ..
